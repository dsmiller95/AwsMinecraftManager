Modpack details

the current modpack is playable with vanilla minecraft, version 1.18.2. the modpack will provide you with a minimap and a visual interface to claim chunks with instead of only a command based interface

the modpack is this zip file (don't worry it doesn't have any .exes in it) to set it up in the curseforge launcher:

  • click Create custom profile
  • in that modal select the "Import a previously created profile option"
  • point it to the zip file and youll be set

Vanilla Chunk claiming guide

All chunk claiming and unclaiming in vanilla is done with the /ftbchunks command

  • TIP: F3 + G will show chunk borders
  • You can claim up to 4 chunks
  • /ftbchunks info : check if chunk is claimed already
  • /ftbchunks claim : claims the chunk you are in
  • /ftbchunks unclaim : unclaims the chunk you are in
  • /ftbchunks unclaim_all : unclaims all your claimed chunks

Allying

  • each of the players forms their own team /ftbteams party create YourCoolTeamNameHere
  • ally your team with your allies' teams /party allies add OtherTeamNameHere
